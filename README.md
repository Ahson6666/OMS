# OMS (Order Management System)
A trading platform for investment banks: order book, matching engine, portfolio & PnL.

Features:
Order book with market/limit order matching (concurrent)
REST APIs for orders, portfolio, PnL
PostgreSQL persistence with JPA
Live price feed simulation
Risk controls & JUnit tests (80%+ coverage)

Tech Stack:
Java 17+
Spring Boot 3.2+
PostgreSQL
GitHub Actions CI

Skills Involved:
Core Java: Concurrency, streams, collections
Enterprise: Spring Boot REST + JPA, SQL design
Banking: Order matching, real-time PnL, risk gates
Production: CI/CD, testing, OpenAPI docs
